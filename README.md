#  DayMate
**Your Smart Daily Companion**

DayMate is a smart app that organizes your entire day in one seamless experience.  
It combines:
- ⏰ Smart Alarm  
- 🧘 Daily Health Tips  
- ✅ Smart To-Do List  
- 🕌 Prayer Reminders  
- ⭐ Favorites Section  

All in a simple, user-friendly interface designed for the Arab world and beyond.
